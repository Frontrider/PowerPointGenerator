Configuração do endereço IP
O dispositivo sem fios tem de ser configurado para utilizar um endereço. Tem duas opções possíveis:

Obter um endereço IP automaticamente. Utilize esta opção se a sua rede estiver a utilizar um servidor DHCP.

Definir endereço IP estático Utilize esta opção se a sua rede não estiver a utilizar um servidor DHCP e quiser introduzir manualmente as definições de endereço IP e de máscara de sub-rede.

Se não tiver a certeza qual a opção que deve selecionar, contacte o seu administrador de sistema.

Nota: se selecionar a opção "Obter um endereço IP automaticamente", o dispositivo tentará adquirir um endereço IP a partir de um servidor DHCP. Se não conseguir comunicar com o servidor DHCP porque este está indisponível ou porque configurou incorretamente as definições de segurança de rede sem fios, o dispositivo mudará para um endereço IP baseado em APIPA, dentro do intervalo 169.254.0.1 a 169.254.255.254.
