Glossário
802.11b ou 802.11g
As redes de 802.11b podem funcionar a um máximo de 11Mbits por segundo. As redes de 802.11g funcionam a até 54 Mbits por segundo.

Ponto de acesso
Também conhecido como router sem fios, um ponto de acesso representa o núcleo da sua rede sem fios, controlando a sua configuração de segurança. Se estiver a utilizar um ponto de acesso, significa que tem uma rede no modo de infraestrutura.

Ad-hoc
Cada dispositivo comunica directamente com outro. Não existe ponto de acesso central a controlar as comunicações da rede sem fios. Este tipo de rede é também designado por rede sem fios ponto-a-ponto.

APIPA
O protocolo Automatic Private IP Addressing (APIPA) é utilizado quando o dispositivo ligado em rede não consegue obter um endereço IP a partir do servidor DHCP.

ASCII
American Standard Code for Information Interchange (ASCII). A representação numérica de um carácter. Por exemplo, o código ASCII para a letra A é 65.

Método de autenticação
O tipo de autenticação para redes sem fios utilizado na sua rede.

Canal
As redes sem fios utilizam canais. Cada canal utiliza uma frequência radioelétrica diferente, dentro de um intervalo pré-determinado. Todos os dispositivos sem fios dentro dessa rede utilizarão o mesmo canal, exceto se esses dispositivos estiverem ligados a outros pontos de acesso sem fios que funcionem como gateway para a rede sem fios.

DHCP
Um servidor Dynamic Host Configuration Protocol (DHCP) atribui endereços IP a dispositivos de rede.

DSL
Digital Subscriber Line (DSL), também conhecido por ADSL ou xDSL. Uma ligação de alta velocidade à Internet.

EAP
Protocolo Extensible Authentication Protocol (EAP), que oferece uma estrutura de autenticação para redes sem fios e com fios. O EAP utiliza um servidor RADIUS, que autentica utilizadores em redes empresariais antes de poderem aceder à rede sem fios ou com fios.

Cabo de rede (cabo Ethernet)
O cabo utilizado para ligar o dispositivo à sua rede com fios. Os cabos Ethernet utilizam conectores RJ45 e o cabo em si é também designado por cabo direto (straight-through).

Endereço de gateway
Um endereço de gateway permite ao dispositivo comunicar com outros dispositivos em diferentes redes. Geralmente, não é necessário configurar um endereço de gateway.

HEX.
Abreviatura de HEXADECIMAL, um sistema numérico que utiliza os números 0-9 e as letras A-F.

Infraestrutura
Cada dispositivo comunica através de um ponto de acesso central. É o tipo de rede sem fios mais habitualmente utilizado.

Endereço IP
Cada dispositivo na rede tem de utilizar um endereço exclusivo. Este endereço é conhecido como um endereço IP e pode ser introduzido manualmente ou ser atribuído automaticamente utilizando um servidor DHCP. O endereço IP é um elemento obrigatório numa rede TCP/IP.

LEAP
Lightweight Extensible Authentication Protocol (LEAP) Um protocolo da Cisco utilizado para autenticação 802.1X sem redes locais (LAN) sem fios. Tal como o EAP, o LEAP utiliza um servidor RADIUS, que autentica utilizadores em redes empresariais antes de poderem aceder à rede sem fios ou com fios.

Endereço MAC
Um endereço Media Access Control (MAC) é um número exclusivo de 12 dígitos atribuído a cada dispositivo de rede.

Nome do nó
Um nome exclusivo para o dispositivo sem fios. Geralmente, o nome do nó baseia-se no endereço MAC.

Potência do sinal
A potência do sinal sem fios entre o dispositivo e o ponto de acesso com o qual está a comunicar.

Subnet Mask
Um endereço IP necessita também de uma máscara de sub-rede. A Subnet Mask é introduzida manualmente ou atribuída automaticamente, utilizando um servidor DHCP. Uma Subnet Mask é um elemento obrigatório numa rede TCP/IP.

SSID
O Service Set Identifier (SSID), também referido como nome da rede, é o nome utilizado pela rede sem fios. Em algumas redes sem fios, o nome SSID não é publicitado, para aumentar a segurança.

Interruptor
Um interruptor permite a comunicação entre vários dispositivos de rede.

TKIP
O protocolo TKIP (Temporal Key Integrity Protocol) remove a previsibilidade de que intrusos poderiam servir-se para explorar as fragilidades ao nível da segurança, devido à utilização de chaves WEP atribuídas manualmente. As chaves TKIP são geradas automaticamente e são mais extensas que as chaves WEP, logo, mais seguras que as chaves WEP padrão.

Chave WEP
A chave Wireless Equivalent Protocol (WEP) é uma funcionalidade de segurança em redes sem fios que protege a rede de eventuais intrusos. Um chave WEP é também designada por Chave de rede.
