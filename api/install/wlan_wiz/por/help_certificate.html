Configuração de Certificado.
Certificado de cliente
Selecione o certificado de cliente para autenticação EAP-TLS. Não se esqueça que primeiro precisa de configurar o Certificado na página "Configurar certificado" a partir da página Web interna (EWS) do dispositivo. (Exceto para EAP-TLS, não tem de selecionar o certificado de cliente quando utiliza um método de autenticação.)
Verificação do certificado do servidor
Selecione o método de verificação do certificado do servidor (não é necessário selecionar o método de verificação quando utiliza LEAP).
  • Sem verificação
    Confia-se no certificado do servidor sem verificação.

  • Cert. CA
    O certificado do servidor emitido pelo CA é verificado pelo certificado CA instalado neste dispositivo. Não se esqueça que primeiro precisa de configurar o Certificado na página "Configurar certificado" a partir da página Web interna (EWS) do dispositivo.

  • Cert. CA + ID do servidor
    São verificados o certificado do servidor (emitido pelo CA utilizando o certificado CA instalado neste dispositivo) e o nome comum (ID do servidor). Não se esqueça que primeiro precisa de configurar o Certificado na página "Configurar certificado" a partir da página Web interna (EWS) do dispositivo.

ID do servidor
Introduza o ID do servidor para a verificação do certificado do servidor. O ID do servidor é o nome comum do certificado do servidor.

Configurar certificado
Se estiver a utilizar um certificado de cliente ou um certificado CA, primeiro precisa de configurar o Certificado na página "Configurar certificado" a partir da página Web interna (EWS) do dispositivo.
