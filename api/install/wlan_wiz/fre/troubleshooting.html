Description du problème :
1. Le périphérique sans fil ne peut pas se connecter au réseau sans fil.

Solution:
SSID
Veuillez vous assurer que vous avez correctement saisi le SSID, aussi appelé nom du réseau, lors de la configuration du périphérique sans fil. Veuillez aussi noter que le SSID fait la distinction entre majuscules et minuscules. Ainsi, si votre SSID est "Réseau Sans fil" et que vous entrez "réseau sans fil" lors de la configuration du périphérique sans fil, ce dernier ne pourra pas se connecter au réseau car le R et le S ne sont pas en majuscules comme dans "Réseau Sans fil".

CLE WEP
Veuillez aussi vous assurer que vous avez entré la clé WEP/TKIP correcte lors de la configuration du périphérique sans fil. Entrez la clé correcte en veillant à saisir de 8 à 63 caractères en fonction de la clé utilisée lors de la configuration du réseau sans fil. Dans les deux cas, si vous avez saisi des informations incorrectes, nous vous recommandons de réinitialiser les paramètres sans fil du périphérique en les ramenant à leurs valeurs par défaut et relancez le logiciel de l’Assistant de configuration de la connexion sans fil. Pour savoir comment réinitialiser les paramètres sans fil du périphérique sans fil, veuillez consulter la documentation accompagnant ce dernier.

Description du problème :
2. Lors de la recherche de réseaux sans fil, l’Assistant n'a pas trouvé le réseau sans fil.

Solution:
Si votre périphérique sans fil se trouve hors de portée de votre réseau sans fil, rapprochez-le du point d’accès sans fil.

Si le périphérique sans fil est à la portée de votre réseau sans fil, vérifiez que le point d’accès sans fil ne masque pas son nom de réseau SSID. Si c’est le cas, vous devez entrer manuellement le nom de réseau SSID lors de la configuration de votre périphérique sans fil. Si vous ne savez pas comment vérifier ceci, veuillez consulter votre administrateur système, votre fournisseur d'accès Internet ou le fabricant de votre point d’accès sans fil.

Description du problème :
3. L’intensité du signal est faible ou parfois nulle.

Solution:
Les périphériques sans fil sont affectés par l’environnement dans lequel ils se trouvent et l’intensité du signal peut être perturbée par le mobilier, les murs ou le sol se trouvant entre le périphérique sans fil et le point d’accès avec qui il essaye de communiquer. Si vous rencontrez des problèmes d’intensité de signal, essayez de rapprocher le périphérique sans fil du point d’accès.

4. (Uniquement applicable aux utilisateurs qui n’emploient pas de câble USB ou LAN pour configurer leur périphérique sans fil). J’ai fait une erreur en configurant les paramètres sans fil mais quand je relance le logiciel de l’Assistant de configuration de la connexion sans fil pour saisir les paramètres corrects, il ne trouve pas de périphérique.

Solution:
Il est possible que le logiciel de l’Assistant ne puisse pas détecter votre périphérique sans fil parce que les paramètres sans fil définis sur votre ordinateur et sur le périphérique sont à présent complètement différents. Essayez de réinitialiser les paramètres sans fil sur le périphérique à leur valeur par défaut et relancez l’Assistant.

Pour en savoir plus sur comment réinitialiser les paramètres sans fil du périphérique à leurs valeurs par défaut en sortie d'usine, veuillez consulter votre guide de l’utilisateur.
