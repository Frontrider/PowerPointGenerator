Configuration de l'adresse IP
Le périphérique sans fil doit être configuré pour utiliser une adresse IP. Vous avez deux possibilités :

Obtenir une adresse IP automatiquement. Choisissez cette option si votre réseau utilise un serveur DHCP.

Définir une adresse IP statique. Sélectionnez cette option si votre réseau n'utilise pas de serveur DHCP et si vous souhaitez entrer manuellement l'adresse IP et les paramètres de masque de sous-réseau.

Si vous ne savez pas quelle option choisir, veuillez demander conseil à votre administrateur système.

Veuillez noter les points suivants : Si vous sélectionnez l'option “Obtenir une adresse IP automatiquement”, votre périphérique essaiera d'acquérir une adresse IP auprès d'un serveur DHCP. S'il ne peut pas communiquer avec le serveur DHCP parce que celui-ci n'est pas disponible ou que vous avez fait une erreur en configurant vos paramètres de sécurité sans fil, le périphérique adoptera une adresse IP APIPA dans la plage 169.254.0.1-169.254.255.254.
