Zertifikatkonfiguration
Client-Zertifikat
Wählen Sie das Client-Zertifikat für die EAP-TLS-Authentifizierung. Bitte beachten Sie, dass Sie zuerst das Zertifikat auf der Seite "Zertifikat konfigurieren" der internen Webseite (EWS) Ihres Gerätes einrichten müssen. (Sie müssen das Client-Zertifikat nicht wählen, wenn Sie eine Authentifizierungsmethode verwenden, außer für EAP-TLS.)
Server-Zertifikat-Verifizierung
Wählen Sie die Methode zur Server-Zertifikat-Verifizierung. (Sie müssen die Methode zur Server-Zertifikat-Verifizierung nicht wählen, wenn Sie LEAP verwenden.)
  • Keine Verifizierung
    Das Server-Zertifikat ist ohne Verifizierung vertrauenswürdig.

  • CA-Zert.
    Das von der Zertifizierungsstelle (CA) ausgestellte Server-Zertifikat wird durch das in diesem Gerät installierte CA-Zertifikat verifiziert. Bitte beachten Sie, dass Sie zuerst das Zertifikat auf der Seite "Zertifikat konfigurieren" der internen Webseite (EWS) Ihres Gerätes einrichten müssen.

  • CA-Zert. + Server-ID
    Das Server-Zertifikat (ausgestellt durch die Zertifizierungsstelle (CA) unter Verwendung des in diesem Gerät installierten CA-Zertifikats) und der allgemeine Name (Server-ID) sind verifiziert. Bitte beachten Sie, dass Sie zuerst das Zertifikat auf der Seite "Zertifikat konfigurieren" der internen Webseite (EWS) Ihres Gerätes einrichten müssen.

Server-ID
Geben Sie die Server-ID für die Server-Zertifikat-Verifizierung ein. Die Server-ID ist der allgemeine Name des Server-Zertifikats.

Zertifikat konfigurieren
Wenn Sie ein Client-Zertifikat oder ein CA-Zertifikat verwenden möchten, muss das Zertifikat zuvor auf der Seite "Zertifikat konfigurieren" der internen Webseite (EWS) Ihres Gerätes eingerichtet werden.
