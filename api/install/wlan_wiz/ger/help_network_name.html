Name des Wireless-Netzwerkes
Geben Sie den Netzwerknamen (SSID) ein, mit dem das Wireless-Gerät verbunden werden soll.

Wird das Gerät in einem Ad-hoc-Netzwerk ohne Access Point genutzt, aktivieren Sie bitte die Option „Dies ist ein Ad-hoc-Netzwerk und es gibt keinen Access Point/Basisstation“. Ist diese Option aktiviert, müssen Sie den verwendeten Netzwerkkanal manuell eingeben. Falls Sie nicht Ihren verwendeten Netzwerkkanal wissen, dann entnehmen Sie bitte der dem Computer mitgelieferten Dokumentation wie Kanäle eingestellt werden.
